## What to build
Create the Staff Dashboard route. Build the form builder interface where Staff can create a Logbook by adding preset fields or custom fields. The resulting schema should be saved to the mocked store.

## Acceptance criteria
- [ ] Staff Dashboard route exists and is protected/rendered for Staff and Admins.
- [ ] UI exists to create a new Logbook.
- [ ] Users can add custom text fields or preset fields to the Logbook configuration.
- [ ] The Logbook configuration is saved to the store.
