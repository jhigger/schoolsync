## What to build
Add a "Launch Kiosk" button to a configured Logbook. Build a specialized Kiosk Route that strictly displays the logbook form and completely hides the App Shell navigation. Implement a PIN-protected "Exit Kiosk" modal to prevent students/guests from escaping.

## Acceptance criteria
- [ ] "Launch Kiosk" button transitions the app to Kiosk Mode.
- [ ] Kiosk route hides all global navigation and sidebars.
- [ ] "Exit Kiosk" button prompts for a PIN/password to return to the Staff Dashboard.
