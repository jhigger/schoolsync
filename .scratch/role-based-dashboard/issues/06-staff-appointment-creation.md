## What to build
Add UI for Staff to create an appointment or requirement for a specific student. Build a management list for Staff to track the status of these appointments (Pending, RSVP'd, Cancelled, Done).

## Acceptance criteria
- [ ] Staff can create an appointment/requirement targeted at a student.
- [ ] Appointment is saved to the store.
- [ ] Staff can view a list of created appointments and their current status.
