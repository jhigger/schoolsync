## What to build
Implement the `/settings` route including user preferences. This includes toggles for Dark Mode and Simple/Detailed mode that hook into global state. Build this using `shadcn/ui` (Switches, Selects, Cards).

## Acceptance criteria
- [ ] `/settings` route is created and accessible.
- [ ] Dark Mode toggle is implemented and functional.
- [ ] Simple/Detailed mode toggle is implemented.
- [ ] Preferences UI is built with shadcn/ui components.
